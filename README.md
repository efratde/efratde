# Dr. Efrat Dener

**Plant ecologist · Quantitative & computational researcher**  
Postdoctoral researcher, Ben-Gurion University of the Negev

I study **seed dispersal, plant trait variation, and community ecology** across spatial scales, using field experiments, statistical modelling, and simulation. My work combines ecological theory with computational methods — hierarchical models, agent-based simulations, image analysis, and interactive data tools.
